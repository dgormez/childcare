# Daily Care Event Workflow

## Purpose

Capture what happens during a child's day and communicate it appropriately.

### Trigger

A caregiver observes a child event.

Examples:

- Nap completed.
- Meal eaten.
- Activity completed.
- Photo captured.

### Actors

- Caregiver
- Parent
- Director
- System

### Flow

1. Caregiver records event.
2. System validates information.
3. Event attaches to child's timeline.
4. Parent receives appropriate update.
5. Director reporting may use aggregated information.

### Applications

Caregiver Tablet:

- Fast event entry.

Parent Mobile:

- View child's day.

Director Web:

- Review trends.

### Design Principles

- Recording should take seconds.
- Information should feel human.
- Parents see a story, not database records.
