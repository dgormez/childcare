# Attendance Workflow

## Purpose

Allow childcare staff to know which children are present and responsible for them at any
moment.

### Trigger

A child arrives or leaves childcare.

### Actors

- Parent
- Caregiver
- Director
- System

### Flow

1. Parent brings child to center.
2. Caregiver confirms arrival.
3. System creates attendance record.
4. Child status changes to present.
5. Parent receives confirmation if enabled.
6. Director reporting updates.

### Applications

Caregiver Tablet:

- View expected arrivals.
- Mark arrival.
- Mark departure.
- Correct mistakes.

Parent Mobile:

- View attendance status.

Director Web:

- Monitor attendance.
- Review history.

### Data

Attendance record:

- Child.
- Classroom.
- Date.
- Arrival time.
- Departure time.
- Caregiver.
- Status.
